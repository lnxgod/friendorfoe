# Friend or Foe — د الوتکو او ډرونونو ریښتیني وخت پیژندنه

[![Android Build](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml/badge.svg)](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml)

**خپل ټیلیفون اسمان ته ونیسئ. وپوهیږئ چې هلته څه شی دی.**

Friend or Foe یو خلاص سرچینې، **نصب-او-پیل** Android اپلیکیشن دی چې الوتکې او ډرونونه د واقعیت زیاتولو (AR) په کارولو سره په ریښتیني وخت کې پیژني. دا د ADS-B ټرانسپونډر ډیټا، د FAA Remote ID ډرون خپرونې، WiFi سیګنال تحلیل، او د وسیلې په کچه بصري کشف سره یوځای کوي ترڅو د کامرې لید باندې تیرېدونکي لیبلونه واچوي — تاسو ته وايي چې کوم شی پورته الوتنه کوي، چا یې چلوي، چیرته ځي، او ایا دوست دی که دښمن. هیڅ حساب، هیڅ ثبت نام، هیڅ API کیلي ته اړتیا نشته — یوازې نصب کړئ او پیل کړئ.

دا پروژه **د AI سره جوړه شوې** — نه یوازې د یو سره، بلکې د ټولو سره. Claude کوډ لیکلی، Grok ډیزاین جوړ کړی، Codex امنیت بیاکتنه کړې، او Gemini د ټیکنالوژۍ سټیک غوره کولو کې مرسته کړې. د [GAMECHANGERSai](https://gamechangersai.org) لخوا خپور شوی ترڅو وښیي چې کله AI د انسان خلاقیت سره سره وکارول شي څه ممکن دي. د نسخو تاریخ لپاره [CHANGELOG](CHANGELOG.md) وګورئ.

### د ۷۲ ساعتو چټک سیالي

> **له صفر څخه تر تایید شوي الوتکو او ډرونونو کشف پورې د ریښتیني وسیلې باندې — د ۷۲ ساعتو لاندې.**
>
> د مارچ ۱۲، ۲۰۲۵ کال، لومړنی کمیټ هماغه ماښام ترسره شو، ۱۳ کمیټونه او **۸,۵۰۰+ لینونه کوډ** د Claude سره د AI ګډ پروګرامینګ په ~۲ ساعتو کې لیکل شوي — چې یو د جوړولو لپاره چمتو APK د AR لید، ډیری سرچینو کشف، Bayesian سینسر فیوژن، او نقشې لید سره تولید کړ. تر مارچ ۱۴ پورې، اپلیکیشن د الوتکو سیلوېتونو، طرزدار نقشې نښانونو، او صافکارۍ سره د خلاصې سرچینې په توګه خپور شو — ټول **۲۲,۰۰۰+ لینونه** د Kotlin، Python، او XML پورې.
>
> اپلیکیشن **نصب-او-پیل** دی — APK نصب کړئ، اجازې ورکړئ، او د الوتکو پیژندنه پیل کړئ. دا مستقیماً وړیا عامه ADS-B APIانو سره وصلیږي پرته له سرور، پرته له کیلیو، پرته له حسابونو. اختیاري Python بیکینډ یوازې اضافي غنیمت (د هوایي شرکت نومونه، د مسیر معلومات) زیاتوي.

## څه کوي

- **AR لید** — ژوندی کامرې لید د کشف شویو الوتکو او ډرونونو باندې د رنګ کوډ شوي تیرېدونکو لیبلونو سره. لیبلونه کالساین، د الوتکې ډول، لوړوالی، او واټن ښیي.
- **ډیری سرچینو کشف** — د هرڅه خبرتیا لپاره څلور خپلواکې کشف میتودونه یوځای کوي:
  - ADS-B ټرانسپونډر ډیټا (سوداګریز الوتنې، عمومي هوایي ترانسپورت)
  - FAA Remote ID د Bluetooth LE له لارې (اطاعت کونکي ډرونونه)
  - WiFi SSID نمونه تطابق (DJI، Skydio، Parrot، ۱۰۰+ جوړونکي)
  - بصري کشف د ML Kit سره (د کامرې پر بنسټ شی پیژندنه)
- **هوښیار طبقه بندي** — هرڅه په ۱۰ ډولونو طبقه بندي کوي: سوداګریز، عمومي هوایي ترانسپورت، پوځي، هلیکوپټر، دولتي، بیړني، بار، ډرون، ځمکنی واسطه، او نامعلوم. پوځي کشف د کالساین نمونو، سکواک کوډونو، او د چلونکو ډیټابیسونو څخه کار اخلي.
- **Bayesian سینسر فیوژن** — کله چې ډیری سینسرونه ورته شی کشف کوي، د باور نمرې د Bayesian log-odds په کارولو سره یوځای کیږي — نه یوازې "تر ټولو لوړ غوره کړئ." دوه ضعیف سیګنالونه چې موافقت ولري کولی شي د یو قوي سیګنال څخه ډیر اغیز ولري.
- **د الوتکو سیلوېتونه او عکسونه** — ۱۲۰+ ICAO ډول کوډونه د ۱۰ ویکتور سیلوېت کتګوریو سره نقشه شوي (تنګ بدن، پلن بدن، سیمه ایز، ټربوپراپ، بزنس جیټ، هلیکوپټر، جنګي الوتکه، بار، سپک الوتکه، ډرون) او ۱۳۴ بنډل شوي الوتکو عکسونه د فوري بصري پیژندنې لپاره — شبکې ته اړتیا نشته.
- **2D نقشې لید** — OpenStreetMap د هرې کتګورۍ لپاره جلا نښان شکلونو، واټن حلقو، قطب نما تعقیب حالت، او FOV مخروط پوښښ سره.
- **تفصیل کارتونه** — د بشپړ تفصیلاتو لپاره هر شی ټپ کړئ: ثبت، چلونکی، مسیر (اصلي/منزل)، لوړوالی، سرعت، سمت، سکواک کوډ، د کشف سرچینه، او د باور کچه.
- **تاریخ لاګ** — د هرڅه چې تاسو پیژندلي د لټون وړ او ترتیب وړ دایمي ډیټابیس.
- **د ډرون حوالې لارښود** — د عکسونو، مشخصاتو، او توضیحاتو سره د ۳۰+ ډرون ډولونو داخلي ډیټابیس — د DJI مصرف کونکي ډرونونو څخه تر پوځي UAVانو لکه Bayraktar TB2، MQ-9 Reaper، او Shahed-136 پورې. د کتګورۍ له مخې لټون وکړئ، د نوم سره وپلټئ، یا مستقیماً د نامعلوم ډرون کشف څخه وپوهیږئ چې تاسو څه ګورئ.
- **بیکینډ غنیمت (اختیاري)** — یو اختیاري Python API سرور کولی شي د هوایي شرکتونو نومونه، د ثبت شمیرې، او د لا دمخه بنډل شوو څخه اضافي مسیر معلومات زیات کړي. اپلیکیشن پرته له دې بشپړ کار کوي — د الوتکو عکسونه او ډرون حوالې APK کې شامل دي.

## څنګه کار کوي

### د کشف سرچینې

| سرچینه | څه کشفوي | واټن | باور |
|--------|-----------|------|------|
| **ADS-B** | سوداګریزې او عمومي هوایي الوتکې د ټرانسپونډرونو سره | ~۲۵۰ NM | ډیر لوړ (0.95) |
| **Remote ID (BLE)** | د FAA اطاعت کونکي ډرونونه (۲۵۰g+) | ~۳۰۰m | لوړ (0.85) |
| **WiFi SSID** | DJI، Skydio، Parrot، ارزانه ډرونونه | ~۱۰۰m | منځنی (0.3-0.85) |
| **بصري (ML Kit)** | هرڅه چې په کامره کې لیدل کیږي | د لید کرښه | متغیر |

### سینسر فیوژن او AR

اپلیکیشن د اکسلرومیټر، مقناطیسومیټر، او ګایروسکوپ ډیټا فیوز کوي ترڅو معلومه کړي چې ټیلیفون دقیقاً چیرته اشاره کوي. د الوتکو موقعیتونه (lat/lon/لوړوالی) د haversine فاصلې محاسبو او د کامرې FOV جیومیټري په کارولو سره د کامرې لید باندې پروجکت کیږي.

**ARCore + قطب نما هایبریډ**: ARCore عالي تعقیب چمتو کوي کله چې کامره ځمکنۍ ځانګړتیاوې وویني، مګر مبارزه کوي کله چې بې ځانګړتیاوو اسمان ته اشاره وشي — دقیقاً هغه وخت چې تاسو ورته اړتیا لرئ. اپلیکیشن په اوتومات ډول قطب نما ریاضیاتي لورښوونې ته ځي کله چې ARCore تعقیب له لاسه ورکوي، په ټولو حالاتو کې بې درزه لیبلونه چمتو کوي.

## پیل کول

### اړتیاوې

- **Android Studio** Hedgehog (2023.1.1) یا وروسته
- **JDK 17**
- **Android SDK 35** (compileSdk) د minSdk 26 سره (Android 8.0+)
- یو Android وسیله د GPS، قطب نما، او کامرې سره (ایمولیټر د لیست/نقشې لیدونو لپاره کار کوي مګر AR لپاره نه)

> **حساب یا ثبت نام ته اړتیا نشته.** اپلیکیشن مستقیماً وړیا عامه ADS-B APIانو سره وصلیږي — هیڅ بیکینډ تنظیم، هیڅ API کیلي، هیڅ حسابونه.

### ګړندی پیل — یوازې Android

اپلیکیشن مستقیماً وړیا عامه ADS-B APIانو سره وصلیږي — هیڅ بیکینډ تنظیم یا حساب ته اړتیا نشته. د الوتکو عکسونه او د ډرون حوالې لارښود APK کې بنډل شوي. یوازې جوړولو او چلولو لپاره:

```bash
cd android
./gradlew assembleDebug
# په وصل شوي وسیله باندې نصب کول:
adb install app/build/outputs/apk/debug/app-debug.apk
```

یا وروستی دمخه جوړ شوی APK له [**GitHub Releases**](https://github.com/lnxgod/friendorfoe/releases) څخه ډاونلوډ کړئ.

### بیکینډ تنظیم (اختیاري — غنیمت فعالوي)

بیکینډ د الوتکو عکسونه، د هوایي شرکتونو نومونه، د ثبت لټونونه، او د مسیر معلومات زیاتوي. **Python 3.11+ او اختیاری Redis د کیشینګ لپاره اړتیا ده.**

```bash
cd backend

# مجازي چاپیریال جوړ کړئ
python -m venv .venv
source .venv/bin/activate  # یا .venv\Scripts\activate په Windows باندې

# اړتیاوې نصب کړئ
pip install -r requirements.txt

# د چاپیریال تنظیمات کاپي کړئ
cp .env.example .env
# .env سم کړئ که تاسو غواړئ د OpenSky اسناد اضافه کړئ (اختیاري، د نرخ محدودیتونه زیاتوي)

# سرور چل کړئ
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**د Docker سره** (Redis + PostgreSQL شامل دي):

```bash
cd backend
docker compose up
```

API به په `http://localhost:8000` کې شتون ولري. د روغتیا معاینه: `http://localhost:8000/health`

### اپلیکیشن خپل بیکینډ سره وصلول

د Android اپلیکیشن د شبکې تنظیماتو کې د بیکینډ URL تازه کړئ ترڅو خپل سرور IP پتې ته اشاره وکړي.

## د AI سره جوړ شوی

دا پروژه یوازې د یو AI سره نه ده جوړه شوې — دا د **ټولو** سره جوړه شوې. موږ هره اصلي AI پلاتفارم وازمایله او هره یوه هلته وکاروله چیرته چې تر ټولو قوي وه. د git تاریخ کیسه بیانوي:

| نیټه | څه وشول |
|------|---------|
| **مارچ ۱۲** | ۱۳ کمیټونه رسیدلي — **۸,۵۰۰+ لینونه کوډ** د AI ګډ پروګرامینګ په ~۲ ساعتو کې لیکل شوي. پایله: د جوړولو لپاره چمتو APK د AR لید، څلور کشف سرچینو، Bayesian سینسر فیوژن، نقشې لید، او تاریخ تعقیب سره. |
| **مارچ ۱۴** | د خلاصې سرچینې خپرول. ۱۲۰+ الوتکو سیلوېتونه، طرزدار نقشې نښانونه د کتګورۍ شکلونو سره، د اجازو سمون، او امنیتي بیاکتنه اضافه شوه. وروسته: ۱۳۴ الوتکو عکسونه د آفلاین شتمنیو په توګه بنډل شوي، د ښه راغلاست سکرین د تازه معاینه سره، او Coil عکس بارول. |
| **ټول** | **۲۳,۰۰۰+ لینونه** Kotlin، Python، او XML. د ریښتیني نړۍ کې د سوداګریزو الوتکو او ډرونونو کشف په وسیله باندې تایید شوی. |

**vibe coding څه شی دی؟** دا د سافټویر پراختیا لپاره یو ګډکاري، خبرو اترو پر بنسټ طریقه ده چیرته چې انسان او AI په ریښتیني وخت کې یوځای جوړوي. د هرې لین په لاس لیکلو پرځای، تاسو هغه څه توضیح کوئ چې غواړئ، نظرونو باندې تکراروئ، یوځای ډیبګ کوئ، او AI ته پریږدئ چې بنسټیز کار وکړي پداسې حال کې چې تاسو لید او معمارۍ باندې تمرکز کوئ. دا د احساس سره پروګرامینګ دی — او کار کوي.

Friend or Foe د ډیری AIانو سره جوړ شوی، هر یو هلته مرسته کوله چیرته چې ترټولو ښه وه. Claude د اصلي کوډینګ ملګري په توګه خدمت وکړ — له لومړني معمارۍ څخه تر سینسر فیوژن الګوریتمونو، Bayesian ریاضیاتو، او ویکتور ډراوینګ هنر پورې. Grok د ډیزاین لارښوونې کې مرسته وکړه. Codex د خلاصې سرچینې کولو دمخه امنیتي تفتیش ترسره کړ. Gemini تحقیق وکړ چې کومې ټیکنالوژیانې او چوکاټونه باید وکارول شي. او ML Kit په وسیله باندې چلیږي، بصري کشف پرته له کلاوډ انحصار څخه ځواکمن کوي. پدې ذخیره ګاه کې هره فایل د انسان-AI همکارۍ له خوا جوړه شوې.

**موږ باور لرو چې د سافټویر پراختیا راتلونکی د سم کار لپاره سم AI کارول دي**، او موږ دا پروژه خلاصه سرچینه کوو ترڅو تاسو وویني چې دا په عمل کې څنګه ښکاري.

## د GAMECHANGERSai په اړه

**[GAMECHANGERSai](https://gamechangersai.org)** یو 501(c)(3) غیر انتفاعي ادارې ده چې ماشومانو او کورنیو لپاره عملي، AI ځواکمن زده کړې تجربو ته وقف شوې.

> *"لوبې AI لوبې چې هر زده کونکي ته د جوړولو او بیا تنظیمولو زده کړه ورکوي."*

زموږ د پلاتفارم له لارې، زده کونکي ریښتیني، قابل انتقال مهارتونه رامنځته کوي:

- **خلاقه کوډینګ** — د AI سره د مرستیال په توګه فعال پروګرامونه جوړول
- **سیسټمي فکر** — پوهیدل چې پیچلې برخې څنګه اړیکې لري او څنګه یې ډیبګ کول
- **د سرچینو مدیریت** — ستراتیژیکه پلان جوړونه او اصلاح
- **اخلاقي استدلال** — زده کول چې "ایا باید وکړو؟" د "ایا کولی شو؟" سره یوځای وپوښتئ کله چې AI کاروئ

**Friend or Foe دلیل دی چې دا مهارتونه ریښتیني دي.** ورته خلاقه کوډینګ، سیسټمي فکر، او AI همکاري چې ماشومان زموږ پلاتفارم باندې زده کوي د دې بشپړ فعال الوتکو پیژندنې سیسټم جوړولو لپاره وکارول شوه. هغه څه چې د لوبې په توګه پیل کیږي وسیله کیږي. هغه څه چې د لوبې په توګه پیل کیږي تخصص کیږي.

موږ د خلاصې سرچینې ټولنې ته بیرته ورکولو ته ژمن یو. د Friend or Foe عامه خپرولو سره، موږ هیله لرو:

- **هڅونه** ورکړو پراختیا کونکو او زده کونکو ته چې وپلټي چې د AI مرستې پراختیا سره څه ممکن دي
- **وښیو** چې AI خلاق ملګری دی، بدیل نه — پدې اپلیکیشن کې هره پریکړه د انسان قضاوت لخوا لارښوونه شوې
- **وښیو** چې هغه مهارتونه چې زموږ ماشومان زده کوي یوازې د لوبو لپاره نه دي — دا د ریښتیني، ګټورې ټیکنالوژۍ جوړولو بنسټ دی

زموږ ماموریت په اړه د نورو معلوماتو لپاره **[gamechangersai.org](https://gamechangersai.org)** ته مراجعه وکړئ.

## د ډیټا سرچینې او وصفونه

Friend or Foe ۱۰۰٪ وړیا دی — لاندې هره ډیټا سرچینه خلاصه ده او **هیڅ API کیلیو یا پیسه ورکوونکو حسابونو** ته اړتیا نلري.

### ADS-B د الوتکو موقعیت ډیټا

اپلیکیشن درې کچه فال بیک سلسله کاروي ترڅو تاسو تل د الوتکو ډیټا ترلاسه کړئ حتی که یوه سرچینه بنده وي:

| لومړیتوب | سرچینه | API پای ټکی | څه چمتو کوي | تایید |
|----------|--------|-------------|-------------|-------|
| ۱م | **[adsb.fi](https://adsb.fi)** | `opendata.adsb.fi/api` | د ریښتیني وخت الوتکو موقعیتونه، کالساینونه، لوړوالی، سرعت، سمت، سکواک کوډونه | وړیا، کیلي نشته |
| ۲م | **[airplanes.live](https://airplanes.live)** | `api.airplanes.live` | د adsb.fi سره ورته ډیټا (مطابق ADSBx v2 فارمیټ) | وړیا، کیلي نشته |
| ۳م | **[OpenSky Network](https://opensky-network.org)** | `opensky-network.org/api` | ICAO حالت ویکتورونه — موقعیت، سرعت، سمت، عمودي نرخ | وړیا، کیلي نشته (اختیاري حساب د لوړ نرخ محدودیتونو لپاره) |

دا درې واړه سرچینې د نړیوالو ریسیور شبکو څخه ژوندی ADS-B ټرانسپونډر ډیټا چمتو کوي. اپلیکیشن د GPS مختصاتو او واټن سره پوښتنه کوي، او د ناکامۍ یا وخت ختمیدو په صورت کې په اوتومات ډول بلې سرچینې ته ځي.

### غنیمت او ملاتړي ډیټا

| سرچینه | څه چمتو کوي | تایید |
|--------|-------------|-------|
| **[Planespotters.net](https://planespotters.net)** | د ICAO هکس کوډ سره د الوتکو عکسونه — د تفصیل کارتونو باندې ښودل کیږي | وړیا، کیلي نشته |
| **[Open-Meteo](https://open-meteo.com)** | اوسنی هوا (د ورېځو پوښښ، د باد سرعت، حالات) — د کشف پیرامیټرو تنظیمولو لپاره کارول کیږي | وړیا، کیلي نشته |
| **[OpenStreetMap](https://openstreetmap.org)** | د 2D نقشې لید لپاره د OSMDroid له لارې نقشې ټایلونه | وړیا، کیلي نشته |

### د وسیلې باندې کشف (پرته له بهرنیو APIانو)

دا کشف میتودونه بشپړ په ټیلیفون باندې پرته له شبکې تلیفونونو څخه چلیږي:

- **FAA Remote ID (Bluetooth LE)** — د ~۳۰۰m دننه د اطاعت کونکو ډرون خپرونو لپاره سکین کوي
- **WiFi SSID تطابق** — DJI، Skydio، Parrot، او ۱۰۰+ نور ډرون جوړونکي د WiFi نمونو سره پیژني
- **بصري کشف (ML Kit)** — د کامرې پر بنسټ شی پیژندنه چې په وسیله باندې چلیږي
- **پوځي طبقه بندي** — د کالساین نمونې، سکواک کوډونه، او د چلونکو ډیټابیس (ټول په محلي توګه بنډل شوي)
