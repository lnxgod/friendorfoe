Important Notes

  • Power: Each board powered via USB-C. Scanner and uplink share GND but NOT 3.3V (each has its own regulator).
  • UART: Cross-connect TX→RX. Scanner TX goes to Uplink RX. 921600 baud, 8N1.
  • OLED: I2C at address 0x3C, 100kHz. Connect to Uplink board only (GPIO4=SDA, GPIO5=SCL).
  • WiFi: Uplink connects to your network and POSTs to the backend. Scanner has no WiFi STA.
  • C5 UART pins differ: C5 uses GPIO4/5 for UART (not 17/18 like S3). Same uplink pins (GPIO20/21).
  • Backend URL: Configurable at runtime via serial config or NVS. Default: http://192.168.42.145:8000
