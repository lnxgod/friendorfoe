Friend or Foe — 3-Board Sensor Node

Each node: ESP32-S3 (BLE) + ESP32-C5 (WiFi) + ESP32 OLED (Uplink)

Architecture

ESP32-S3 ESP32 + OLED Backend BLE Scanner ──UART1──▶ Uplink Board ──WiFi──▶ FastAPI (Remote ID) GPIO25 (0.96" OLED) :8000 ▲ ESP32-C5 ──UART2──────┘ WiFi Scanner GPIO32 (Promiscuous) Per node: 3 boards, 4 wires (2x UART TX→RX + 2x GND) OLED is built into the uplink board (no extra wiring)

Wiring — One Node

BLE Scanner (ESP32-S3) → Uplink (ESP32 OLED)

ESP32-S3 DevKitC — BLE Remote ID Scanner
Firmware: pio run -e scanner -t upload
UART1 — 921600 baud
ESP32-S3ESP32 OLED (Uplink)Wire
GPIO17 (TX)→GPIO25 (UART1 RX)blue wire
GND→GNDblack wire

WiFi Scanner (ESP32-C5) → Uplink (ESP32 OLED)

ESP32-C5 DevKitC — WiFi Promiscuous Scanner
Firmware: pio run -e scanner-c5 -t upload
UART2 — 921600 baud
ESP32-C5ESP32 OLED (Uplink)Wire
GPIO4 (TX)→GPIO32 (UART2 RX)red wire
GND→GNDblack wire

Uplink Board (ESP32 with 0.96" OLED)

Plain ESP32 + Built-in SSD1306 OLED
Firmware: pio run -e uplink-esp32 -t upload
OLED: SDA=GPIO4, SCL=GPIO15, RST=GPIO16 (on-board, no wiring needed)

OLED Display

fof_node_1 ─────────────── BLE:OK WiFi:OK Svr:OK Up:42 Drones:3 Bat:85% 12:34 * TRACKING

Shows: node ID, BLE scanner status, WiFi scanner status, backend connection, upload count, drone count, battery, uptime

Flash Commands

# ── 1. Flash BLE Scanner (ESP32-S3) ───────────────── cd esp32/scanner pio run -e scanner -t upload # ── 2. Flash WiFi Scanner (ESP32-C5) ──────────────── cd esp32/scanner pio run -e scanner-c5 -t upload # ── 3. Flash Uplink (ESP32 OLED board) ────────────── cd esp32/uplink pio run -e uplink-esp32 -t upload # ── 4. Configure Uplink (serial, 3-sec boot window) ─ FOF_SET:device_id=fof_node_1 FOF_SET:wifi_ssid=YourNetwork FOF_SET:wifi_pass=YourPassword FOF_SET:backend_url=http://192.168.42.145:8000 FOF_SAVE # ── 5. Register Node on Backend ───────────────────── curl -X POST http://192.168.42.145:8000/nodes \ -H "Content-Type: application/json" \ -d '{"device_id":"fof_node_1","name":"North","lat":YOUR_LAT,"lon":YOUR_LON}'

Important Notes

  • Only 2 wires per scanner — TX→RX + GND. No power sharing needed (each board powered via its own USB).
  • S3 TX=GPIO17 connects to uplink GPIO25 (UART1 RX)
  • C5 TX=GPIO4 connects to uplink GPIO32 (UART2 RX)
  • OLED is on-board — no external wiring needed for the display
  • GPS removed from uplink — nodes use fixed positions registered in the backend via POST /nodes
  • All 3 boards powered via USB — each has its own regulator, just share GND
