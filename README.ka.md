# Friend or Foe — თვითმფრინავებისა და დრონების რეალურ დროში იდენტიფიკაცია

[![Android Build](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml/badge.svg)](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml)

**მიმართეთ ტელეფონი ცისკენ. გაიგეთ, რა არის იქ.**

Friend or Foe არის ღია კოდის, **დააინსტალირე-და-იმუშავე** Android აპლიკაცია, რომელიც გაძლიერებული რეალობის გამოყენებით რეალურ დროში ამოიცნობს თვითმფრინავებს და დრონებს. იგი აერთიანებს ADS-B ტრანსპონდერის მონაცემებს, FAA Remote ID დრონის მაუწყებლობას, WiFi სიგნალის ანალიზს და მოწყობილობაზე ვიზუალურ ამოცნობას, რათა კამერის ხედზე მცურავი ლეიბლები გამოჩნდეს — გეტყვით, რა დაფრინავს თქვენს თავზე, ვინ მართავს მას, სად მიდის და მეგობარია თუ მტერი. ანგარიშების, რეგისტრაციის ან API გასაღებების გარეშე — უბრალოდ დააინსტალირეთ და დაიწყეთ.

ეს პროექტი **ხელოვნური ინტელექტით აშენდა** — არა მხოლოდ ერთით, არამედ ყველათი. Claude-მ დაწერა კოდი, Grok-მა ჩამოაყალიბა დიზაინი, Codex-მა განიხილა უსაფრთხოება, ხოლო Gemini-მ დაეხმარა ტექნოლოგიური სტეკის არჩევაში. გამოცემულია [GAMECHANGERSai](https://gamechangersai.org)-ის მიერ, რათა აჩვენოს, რა არის შესაძლებელი, როდესაც ხელოვნური ინტელექტი ხვდება ადამიანურ შემოქმედებას. ვერსიების ისტორია იხილეთ [CHANGELOG](CHANGELOG.md)-ში.

### 72-საათიანი სპრინტი

>

## რას აკეთებს

- **AR მნახველი** — ცოცხალი კამერის ხედი ფერით კოდირებული მცურავი ლეიბლებით ამოცნობილ თვითმფრინავებსა და დრონებზე. ლეიბლები აჩვენებს საძახებელ ნიშანს, თვითმფრინავის ტიპს, სიმაღლეს და მანძილს.
- **მრავალწყაროიანი ამოცნობა** — ყოვლისმომცველი ცის მონიტორინგისთვის ოთხი დამოუკიდებელი ამოცნობის მეთოდის შეთავსება:
  - ADS-B ტრანსპონდერის მონაცემები (კომერციული ფრენები, ზოგადი ავიაცია)
  - FAA Remote ID Bluetooth LE-ით (შესაბამისი დრონები)
  - WiFi SSID ნიმუშების შედარება (DJI, Skydio, Parrot, 100+ მწარმოებელი)
  - ვიზუალური ამოცნობა ML Kit-ით (კამერაზე დაფუძნებული ობიექტის ამოცნობა)
- **ჭკვიანი კლასიფიკაცია** — ყველაფერს 10 ტიპად ანაწილებს: კომერციული, ზოგადი ავიაცია, სამხედრო, ვერტმფრენი, სამთავრობო, საგანგებო, სატვირთო, დრონი, მიწის ტრანსპორტი და უცნობი. სამხედრო ამოცნობა იყენებს საძახებელი ნიშნის ნიმუშებს, squawk კოდებს და ოპერატორთა მონაცემთა ბაზებს.
- **ბაიესის სენსორული შერწყმა** — როდესაც რამდენიმე სენსორი ამოიცნობს ერთსა და იმავე ობიექტს, ნდობის ქულები ერთიანდება ბაიესის log-odds-ის გამოყენებით — არა მხოლოდ "აირჩიე ყველაზე მაღალი." ორი სუსტი სიგნალის თანხმობამ შეიძლება გადაწონოს ერთი ძლიერი სიგნალი.
- **თვითმფრინავების სილუეტები და ფოტოები** — 120+ ICAO ტიპის კოდი შესაბამისობაშია 10 ვექტორული სილუეტის კატეგორიასთან (ვიწრო ფიუზელაჟი, ფართო ფიუზელაჟი, რეგიონალური, ტურბოპროპი, ბიზნეს ჯეტი, ვერტმფრენი, გამანადგურებელი, სატვირთო, მსუბუქი თვითმფრინავი, დრონი) პლუს 134 ჩაშენებული თვითმფრინავის ფოტო დაუყოვნებლივი ვიზუალური ამოცნობისთვის — ქსელის გარეშე.
- **2D რუკის ხედი** — OpenStreetMap თითოეული კატეგორიისთვის განსხვავებული მარკერის ფორმებით, მანძილის რგოლებით, კომპასის თვალყურის დევნების რეჟიმით და FOV კონუსის გადაფარვით.
- **დეტალური ბარათები** — შეეხეთ ნებისმიერ ობიექტს სრული ინფორმაციისთვის: რეგისტრაცია, ოპერატორი, მარშრუტი (წარმოშობა/დანიშნულება), სიმაღლე, სიჩქარე, კურსი, squawk კოდი, ამოცნობის წყარო და ნდობის დონე.
- **ისტორიის ჟურნალი** — ყველაფრის მუდმივი მონაცემთა ბაზა, რაც თქვენ ამოიცანით, ძიების და დახარისხების შესაძლებლობით.
- **დრონების საცნობარო გზამკვლევი** — 30+ დრონის ტიპის ჩაშენებული მონაცემთა ბაზა ფოტოებით, სპეციფიკაციებით და აღწერებით — DJI-ის სამომხმარებლო დრონებიდან Bayraktar TB2, MQ-9 Reaper და Shahed-136 სამხედრო უპილოტო საფრენ აპარატებამდე. დაათვალიერეთ კატეგორიის მიხედვით, მოძებნეთ სახელით, ან პირდაპირ გადადით უცნობი დრონის ამოცნობიდან იმის იდენტიფიცირებაზე, რასაც ხედავთ.
- **ბექენდის გამდიდრება (არჩევითი)** — არჩევითი Python API სერვერი შეუძლია დაამატოს ავიახაზების სახელები, რეგისტრაციის ნომრები და მარშრუტის ინფორმაცია. აპლიკაცია სრულად მუშაობს მის გარეშე — თვითმფრინავების ფოტოები და დრონების საცნობარო მასალა შეტანილია APK-ში.

## როგორ მუშაობს

### ამოცნობის წყაროები

| წყარო | რას ამოიცნობს | რადიუსი | ნდობა |
|-------|-------------|---------|-------|
| **ADS-B** | ტრანსპონდერიანი კომერციული და GA თვითმფრინავები | ~250 NM | ძალიან მაღალი (0.95) |
| **Remote ID (BLE)** | FAA-ს შესაბამისი დრონები (250გ+) | ~300მ | მაღალი (0.85) |
| **WiFi SSID** | DJI, Skydio, Parrot, ბიუჯეტური დრონები | ~100მ | საშუალო (0.3-0.85) |
| **ვიზუალური (ML Kit)** | ნებისმიერი, რაც კამერაში ჩანს | ხედვის ხაზი | ცვალებადი |

### სენსორული შერწყმა და AR

აპლიკაცია აერთიანებს აქსელერომეტრის, მაგნიტომეტრისა და გიროსკოპის მონაცემებს, რათა ზუსტად განსაზღვროს, სად არის მიმართული ტელეფონი. თვითმფრინავების პოზიციები (განედი/გრძედი/სიმაღლე) კამერის ხედზე პროეცირდება ჰავერსინის მანძილის გამოთვლებისა და კამერის FOV გეომეტრიის გამოყენებით.

**ARCore + კომპასის ჰიბრიდი**: ARCore უზრუნველყოფს შესანიშნავ თვალყურის დევნას, როდესაც კამერა მიწის თვისებებს ხედავს, მაგრამ ცარიელ ცაზე მიმართვისას უჭირს — ზუსტად მაშინ, როდესაც ეს ყველაზე მეტად გჭირდებათ. როდესაც ARCore კარგავს თვალყურს, აპლიკაცია ავტომატურად გადაერთვება კომპას-მათემატიკურ ორიენტაციაზე, ყველა პირობებში უწყვეტი ლეიბლების უზრუნველყოფით.

- **Android Studio** Hedgehog (2023.1.1) ან უფრო ახალი
- **JDK 17**
- **Android SDK 35** (compileSdk) minSdk 26-ით (Android 8.0+)
- Android მოწყობილობა GPS-ით, კომპასით და კამერით (ემულატორი მუშაობს სიის/რუკის ხედებისთვის, მაგრამ არა AR-ისთვის)

### სწრაფი დაწყება — მხოლოდ Android

აპლიკაცია პირდაპირ უკავშირდება უფასო საჯარო ADS-B API-ებს — ბექენდის დაყენების ან ანგარიშების გარეშე. თვითმფრინავების ფოტოები და დრონების საცნობარო გზამკვლევი ჩაშენებულია APK-ში. უბრალოდ ასაგებად და გასაშვებად:

```bash
cd android
./gradlew assembleDebug
# დააინსტალირეთ დაკავშირებულ მოწყობილობაზე:
```
