# Friend or Foe — זיהוי מטוסים ורחפנים בזמן אמת

[![Android Build](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml/badge.svg)](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml)

**כוונו את הטלפון לשמיים. דעו מה נמצא שם למעלה.**

Friend or Foe היא אפליקציית אנדרואיד בקוד פתוח, **התקן-והתחל**, המזהה מטוסים ורחפנים בזמן אמת באמצעות מציאות רבודה. היא משלבת נתוני משדרי ADS-B, שידורי FAA Remote ID לרחפנים, ניתוח אותות WiFi וזיהוי חזותי על המכשיר כדי להציג תוויות צפות על תצוגת המצלמה — ולספר לכם מה טס מעליכם, מי מפעיל את זה, לאן זה הולך, והאם זה ידיד או אויב. ללא חשבונות, ללא הרשמה, ללא מפתחות API — פשוט התקינו והתחילו.

הפרויקט הזה **נבנה עם בינה מלאכותית** — לא רק אחת, אלא כולן. Claude כתב את הקוד, Grok עיצב את הדיזיין, Codex סקר את האבטחה, ו-Gemini עזר לבחור את המערך הטכנולוגי. פורסם על ידי [GAMECHANGERSai](https://gamechangersai.org) כדי להדגים מה אפשרי כשבינה מלאכותית פוגשת יצירתיות אנושית. ראו את [יומן השינויים](CHANGELOG.md) להיסטוריית גרסאות.

### ריצת ה-72 שעות

> **מאפס לזיהוי מאושר של מטוסים ורחפנים על מכשיר אמיתי — תוך פחות מ-72 שעות.**
>
> ב-12 במרץ 2025, ההתחייבות הראשונה בוצעה באותו ערב, 13 התחייבויות ו**יותר מ-8,500 שורות קוד** נכתבו בכ-2 שעות של תכנות זוגי עם AI באמצעות Claude — והניבו APK מוכן לבנייה עם מכוון AR, זיהוי ממקורות מרובים, מיזוג חיישנים בייסיאני ותצוגת מפה. עד ה-14 במרץ, האפליקציה פורסמה בקוד פתוח עם צלליות מטוסים, סמני מפה מעוצבים וליטוש — בסך הכל **יותר מ-22,000 שורות** ב-Kotlin, Python ו-XML.
>
> האפליקציה **התקן-והתחל** — התקינו את ה-APK, אשרו הרשאות והתחילו לזהות מטוסים. היא מתחברת ישירות ל-API ציבוריים וחינמיים של ADS-B ללא שרת, ללא מפתחות, ללא חשבונות. שרת ה-Python האופציונלי מוסיף רק העשרה נוספת (שמות חברות תעופה, מידע על מסלול).

## מה האפליקציה עושה

- **מכוון AR** — תצוגת מצלמה חיה עם תוויות צפות מקודדות צבע על מטוסים ורחפנים שזוהו. התוויות מציגות קוד קריאה, סוג מטוס, גובה ומרחק.
- **זיהוי ממקורות מרובים** — משלבת ארבע שיטות זיהוי עצמאיות למודעות מקיפה של השמיים:
  - נתוני משדרי ADS-B (טיסות מסחריות, תעופה כללית)
  - FAA Remote ID דרך Bluetooth LE (רחפנים תואמים)
  - התאמת דפוסי SSID של WiFi (DJI, Skydio, Parrot, יותר מ-100 יצרנים)
  - זיהוי חזותי עם ML Kit (זיהוי אובייקטים מבוסס מצלמה)
- **סיווג חכם** — מסווג הכל ל-10 סוגים: מסחרי, תעופה כללית, צבאי, מסוק, ממשלתי, חירום, מטען, רחפן, רכב קרקעי ולא ידוע. זיהוי צבאי משתמש בדפוסי קודי קריאה, קודי squawk ומסדי נתוני מפעילים.
- **מיזוג חיישנים בייסיאני** — כשמספר חיישנים מזהים את אותו אובייקט, ציוני הביטחון משולבים באמצעות log-odds בייסיאני — לא רק "בחר את הגבוה ביותר." שני אותות חלשים שמסכימים יכולים לגבור על אות חזק אחד.
- **צלליות ותמונות מטוסים** — יותר מ-120 קודי סוג ICAO ממופים ל-10 קטגוריות צלליות וקטוריות (גוף צר, גוף רחב, אזורי, טורבופרופ, מטוס עסקי, מסוק, מטוס קרב, מטען, מטוס קל, רחפן) ועוד 134 תמונות מטוסים מובנות לזיהוי חזותי מיידי — ללא צורך ברשת.
- **תצוגת מפה דו-ממדית** — OpenStreetMap עם צורות סמן ייחודיות לכל קטגוריה, טבעות מרחק, מצב מעקב מצפן וחיפוי חרוט שדה ראייה.
- **כרטיסי פרטים** — הקישו על כל אובייקט לפרטים מלאים: רישום, מפעיל, מסלול (מוצא/יעד), גובה, מהירות, כיוון, קוד squawk, מקור זיהוי ורמת ביטחון.
- **יומן היסטוריה** — מסד נתונים קבוע של כל מה שזיהיתם, ניתן לחיפוש ומיון.
- **מדריך עיון רחפנים** — מסד נתונים מובנה של יותר מ-30 סוגי רחפנים עם תמונות, מפרטים ותיאורים — מרחפני DJI לצרכנים ועד כלי טיס צבאיים כמו Bayraktar TB2, MQ-9 Reaper ו-Shahed-136. גלשו לפי קטגוריה, חפשו לפי שם, או קפצו ישירות מזיהוי רחפן לא ידוע כדי לזהות מה אתם רואים.
- **העשרת עורפי (אופציונלי)** — שרת API אופציונלי ב-Python יכול להוסיף שמות חברות תעופה, מספרי רישום ומידע נוסף על מסלולים מעבר למה שכבר מובנה. האפליקציה עובדת בצורה מלאה בלעדיו — תמונות מטוסים ומדריך רחפנים כלולים ב-APK.

## איך זה עובד

### מקורות זיהוי

| מקור | מה הוא מזהה | טווח | ביטחון |
|------|-------------|------|--------|
| **ADS-B** | מטוסים מסחריים ותעופה כללית עם משדרים | ~250 NM | גבוה מאוד (0.95) |
| **Remote ID (BLE)** | רחפנים תואמי FAA (250 גרם+) | ~300 מ' | גבוה (0.85) |
| **WiFi SSID** | DJI, Skydio, Parrot, רחפנים חסכוניים | ~100 מ' | בינוני (0.3-0.85) |
| **חזותי (ML Kit)** | כל דבר הנראה במצלמה | קו ראייה | משתנה |

### מיזוג חיישנים ו-AR

האפליקציה ממזגת נתוני מד תאוצה, מגנטומטר וג'ירוסקופ כדי לקבוע בדיוק לאן הטלפון מכוון. מיקומי מטוסים (רוחב/אורך/גובה) מוקרנים על תצוגת המצלמה באמצעות חישובי מרחק haversine וגאומטריית שדה ראייה של המצלמה.

**ARCore + היברידי מצפן**: ARCore מספק מעקב מעולה כשהמצלמה רואה תכונות קרקע, אך מתקשה כשהיא מכוונת לשמיים חסרי תכונות — בדיוק כשצריכים את זה הכי הרבה. האפליקציה עוברת אוטומטית לכיוון מבוסס חישוב מצפן כש-ARCore מאבד מעקב, ומספקת תוויות חלקות בכל התנאים.

## תחילת עבודה

### דרישות מוקדמות

- **Android Studio** Hedgehog (2023.1.1) או מאוחר יותר
- **JDK 17**
- **Android SDK 35** (compileSdk) עם minSdk 26 (Android 8.0+)
- מכשיר אנדרואיד עם GPS, מצפן ומצלמה (אמולטור עובד לתצוגות רשימה/מפה אך לא ל-AR)

> **ללא חשבונות או הרשמה.** האפליקציה מתחברת ישירות ל-API ציבוריים וחינמיים של ADS-B — ללא הגדרת עורפי, ללא מפתחות API, ללא חשבונות.

### התחלה מהירה — אנדרואיד בלבד

האפליקציה מתחברת ישירות ל-API ציבוריים וחינמיים של ADS-B — ללא הגדרת עורפי או חשבונות. תמונות מטוסים ומדריך עיון הרחפנים מובנים ב-APK. כדי פשוט לבנות ולהריץ:

```bash
cd android
./gradlew assembleDebug
# Install on connected device:
adb install app/build/outputs/apk/debug/app-debug.apk
```

או הורידו את ה-APK המוכן האחרון מ-[**GitHub Releases**](https://github.com/lnxgod/friendorfoe/releases).

### הגדרת עורפי (אופציונלי — מאפשר העשרה)

העורפי מוסיף תמונות מטוסים, שמות חברות תעופה, חיפושי רישום ומידע על מסלולים. **דורש Python 3.11+ ואופציונלית Redis לאחסון מטמון.**

```bash
cd backend

# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows

# Install dependencies
pip install -r requirements.txt

# Copy environment config
cp .env.example .env
# Edit .env if you want to add OpenSky credentials (optional, increases rate limits)

# Run the server
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**עם Docker** (כולל Redis + PostgreSQL):

```bash
cd backend
docker compose up
```

ה-API יהיה זמין ב-`http://localhost:8000`. בדיקת תקינות: `http://localhost:8000/health`

### חיבור האפליקציה לעורפי שלכם

עדכנו את כתובת ה-URL של העורפי בתצורת הרשת של אפליקציית האנדרואיד כך שתצביע על כתובת ה-IP של השרת שלכם.

**מה זה vibe coding?** זו גישה שיתופית ודיאלוגית לפיתוח תוכנה שבה אדם ו-AI בונים יחד בזמן אמת. במקום לכתוב כל שורה ביד, אתם מתארים מה אתם רוצים, חוזרים על רעיונות, מאתרים באגים יחד ונותנים ל-AI לטפל בקוד השגרתי בזמן שאתם מתמקדים בחזון ובארכיטקטורה. זה תכנות לפי אווירה — וזה עובד.

Friend or Foe נבנה עם מספר מערכות AI, כל אחת תורמת היכן שהיא מצטיינת. Claude שימש כשותף הקידוד הראשי — מהארכיטקטורה הראשונית דרך אלגוריתמי מיזוג חיישנים, מתמטיקה בייסיאנית ואמנות drawable וקטורית. Grok עזר לעצב את כיוון העיצוב. Codex ביצע את ביקורת האבטחה לפני הפרסום בקוד פתוח. Gemini חקר אילו טכנולוגיות ומסגרות להשתמש. ו-ML Kit רץ על המכשיר, מפעיל זיהוי חזותי ללא תלות בענן. כל קובץ במאגר הזה עוצב על ידי שיתוף פעולה בין אדם ל-AI.

**אנחנו מאמינים שעתיד פיתוח התוכנה הוא שימוש ב-AI הנכון לעבודה הנכונה**, ואנחנו מפרסמים את הפרויקט הזה בקוד פתוח כדי שתוכלו לראות איך זה נראה בפועל.

## מקורות נתונים וייחוס

Friend or Foe הוא חינמי לחלוטין לשימוש — כל מקור נתונים להלן הוא פתוח ואינו דורש **מפתחות API או חשבונות בתשלום**.

### נתוני מיקום מטוסים ADS-B

האפליקציה משתמשת בשרשרת גיבוי בת שלוש שכבות כך שתמיד תקבלו נתוני מטוסים גם אם מקור אחד מושבת:

| עדיפות | מקור | נקודת קצה API | מה הוא מספק | אימות |
|--------|------|---------------|-------------|-------|
| ראשון | **[adsb.fi](https://adsb.fi)** | `opendata.adsb.fi/api` | מיקומי מטוסים בזמן אמת, קודי קריאה, גובה, מהירות, כיוון, קודי squawk | חינם, ללא מפתח |
| שני | **[airplanes.live](https://airplanes.live)** | `api.airplanes.live` | אותם נתונים כמו adsb.fi (פורמט ADSBx v2 תואם) | חינם, ללא מפתח |
| שלישי | **[OpenSky Network](https://opensky-network.org)** | `opensky-network.org/api` | וקטורי מצב ICAO — מיקום, מהירות, כיוון, קצב אנכי | חינם, ללא מפתח (חשבון אופציונלי למגבלות קצב גבוהות יותר) |

שלושת המקורות מספקים נתוני משדרי ADS-B חיים מרשתות מקלטים ברחבי העולם. האפליקציה שולחת שאילתות לפי קואורדינטות GPS ורדיוס, ועוברת אוטומטית למקור הבא בכשלון או פקיעת זמן.

### העשרה ונתונים תומכים

### זיהוי על המכשיר (ללא API חיצוני)

שיטות זיהוי אלו רצות לחלוטין על הטלפון ללא קריאות רשת:

- **FAA Remote ID (Bluetooth LE)** — סורק שידורי רחפנים תואמים בטווח ~300 מ'
- **התאמת WiFi SSID** — מזהה DJI, Skydio, Parrot ויותר מ-100 יצרני רחפנים אחרים לפי דפוסי WiFi
- **זיהוי חזותי (ML Kit)** — זיהוי אובייקטים מבוסס מצלמה הרץ על המכשיר
- **סיווג צבאי** — דפוסי קודי קריאה, קודי squawk ומסד נתוני מפעילים (הכל ארוז מקומית)
