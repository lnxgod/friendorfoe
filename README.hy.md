# Friend or Foe — Օդանավերի և դրոնների իրական ժամանակի նույնականացում

[![Android Build](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml/badge.svg)](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml)

**Ուղղեք ձեր հեռախոսը երկինք։ Իմացեք, ինչ կա այնտեղ։**

Friend or Foe-ը բաց կոդի, **տեղադրեք-և-սկսեք** Android հավելված է, որը ընդլայնված իրականության օգտագործմամբ իրական ժամանակի նույնականացնում է օդանավերն ու դրոնները։ Այն միավորում է ADS-B տրանսպոնդերի տվյալները, FAA Remote ID դրոնների հեռարկումները, WiFi ազդանշանի վերլուծությունը և սարքի վրա տեսողական հայտնաբերումը։ Առանց հաշիվների, առանց գրանցման, առանց API բանալիների — պարզապես տեղադրեք և սկսեք։

Այս նախագիծը **կառուցվել է արհեստական ինտելեկտի** — ոչ թե մեկով, այլ բոլորով։ Claude-ը գրեց կոդը, Grok-ը ձևավորեց դիզայնը, Codex-ը վերանայեց անվտանգությունը, Gemini-ն օգնեց տեխնոլոգիական փաթեթի ընտրությանը։ Թողարկվել է [GAMECHANGERSai](https://gamechangersai.org)-ի կողմից։ Տեսեք [CHANGELOG](CHANGELOG.md)՝ տարբերակի պատմության համար։

### 72-Պատյա արագության վազք

>
> 2025թ. մարտի 12-ին առաջին commit-ը արվեց, 13 commit և **8,500+ տող կոդ** գրվեց ~2 ժամում Claude-ի հետ AI զույգական ծրագրավորմամբ։ 14 մարտին հավելվածը բաց կոդ դարձավ — **22,000+ տող** Kotlin, Python և XML։

## Ինչ է անում

- **AR դիտարկիչ** — Կամերայի ուղիղ տեսք գունավոր լողացող պիտակներով։
- **Բազմացանքի հայտնաբերում** — ADS-B, Remote ID, WiFi SSID, ML Kit։
- **Խելացի դասակարգում** — 10 տեսակի դասակարգում։
- **Բայեսի սենսորի միաձուլում** — Երկու թույլ ազդանշան կարող է գերազանցել մեկին։
- **Օդանավի սիլուետներ և լուսանկարներ** — 120+ ICAO տեսակի կոդեր և 134 լուսանկար։
- **2D քարտեզի դիտման** — OpenStreetMap։
- **Մանրամասն քարտեր** — Սեղմեք ցանկացած օբյեկտի վրա։
- **Պատմության մատյան** — Մշտական տվյալների բազա։
- **Դրոնների տեղեկատվական ուղեցույց** — 30+ դրոնի տեսակներ (Bayraktar TB2, MQ-9 Reaper, Shahed-136)։
- **Սերվերի հարստացում (ընտրելի)** — Ընտրելի Python API սերվեր։

## Ինչպես է աշխատում

| Աղբյուր | Ինչ է հայտնաբերում | Տիրույթ | Վստահություն |
|--------|----------------|-------|-----------|
| **ADS-B** | Տրանսպոնդերով օդանավեր | ~250 NM | Շատ բարձր (0.95) |
| **Remote ID (BLE)** | FAA համապատասխան դրոններ (250գ+) | ~300մ | Բարձր (0.85) |
| **WiFi SSID** | DJI, Skydio, Parrot | ~100մ | Միջին (0.3-0.85) |
| **Տեսողական (ML Kit)** | Կամերայում տեսանելի ամեն ինչ | Տեսանելիության գիծ | Փոփոխական |

## Սկսելու համար

```bash
cd android
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

Կամ ներբեռնեք վերջին APK-ն [**GitHub Releases**](https://github.com/lnxgod/friendorfoe/releases)-ից։
