# Friend or Foe — Ідентифікація повітряних суден та дронів у реальному часі

[![Android Build](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml/badge.svg)](https://github.com/lnxgod/friendorfoe/actions/workflows/android-build.yml)

**Наведіть телефон на небо. Дізнайтеся, що там летить.**

Friend or Foe — це Android-додаток з відкритим кодом, який працює за принципом **встановив-і-готово** та ідентифікує повітряні судна й дрони в реальному часі за допомогою доповненої реальності. Він поєднує дані транспондерів ADS-B, сигнали FAA Remote ID для дронів, аналіз WiFi-сигналів та візуальне розпізнавання на пристрої, щоб накласти плаваючі мітки на зображення камери — повідомляючи вам, що летить над головою, хто це керує, куди воно прямує та чи це друг, чи ворог. Без акаунтів, без реєстрації, без API-ключів — просто встановіть і починайте.

Цей проєкт **створений за допомогою ШІ** — не одного, а всіх. Claude написав код, Grok визначив дизайн, Codex перевірив безпеку, а Gemini допоміг обрати технологічний стек. Випущений [GAMECHANGERSai](https://gamechangersai.org), щоб продемонструвати, що можливо, коли ШІ зустрічає людську креативність. Див. [CHANGELOG](CHANGELOG.md) для історії версій.

>

До 14 березня додаток було опубліковано з відкритим кодом із силуетами літаків, стилізованими маркерами на карті та фінальним полірувванням — загалом **22 000+ рядків** на Kotlin, Python та XML.
>
> Додаток працює за принципом **встановив-і-готово** — встановіть APK, надайте дозволи та починайте ідентифікувати літаки. Він підключається безпосередньо до безкоштовних публічних API ADS-B без реєстрації, без ключів, без акаунтів. Опціональний Python-бекенд лише додає збагачення (назви авіаліній, інформація про маршрут).

## Що він робить

- **AR-видошукач** — Пряма трансляція камери з кольоровими плаваючими мітками на виявлених літаках і дронах. Мітки показують позивний, тип повітряного судна, висоту та відстань.
- **Багатоджерельне виявлення** — Поєднує чотири незалежні методи виявлення для комплексного моніторингу неба:
  - Дані транспондерів ADS-B (комерційні рейси, загальна авіація)
  - FAA Remote ID через Bluetooth LE (сертифіковані дрони)
  - Зіставлення шаблонів WiFi SSID (DJI, Skydio, Parrot, 100+ виробників)
  - Візуальне виявлення з ML Kit (розпізнавання об'єктів на основі камери)
- **Розумна класифікація** — Категоризує все на 10 типів: Комерційний, Загальна авіація, Військовий, Гелікоптер, Урядовий, Екстрений, Вантажний, Дрон, Наземний транспорт та Невідомий. Військове виявлення використовує шаблони позивних, коди squawk та бази даних операторів.
- **Баєсівське злиття сенсорів** — Коли кілька сенсорів виявляють один і той же об'єкт, оцінки впевненості поєднуються за допомогою баєсівських log-odds — а не просто "обрати найвищий". Два слабкі сигнали, що збігаються, можуть переважити один сильний сигнал.
- **Силуети та фотографії літаків** — 120+ кодів типів ICAO зіставлені з 10 категоріями векторних силуетів (вузькофюзеляжний, широкофюзеляжний, регіональний, турбогвинтовий, бізнес-джет, гелікоптер, винищувач, вантажний, легкий літак, дрон) плюс 134 вбудовані фотографії літаків для миттєвого візуального розпізнавання — без мережі.
- **2D-вид карти** — OpenStreetMap з унікальними формами маркерів за категоріями, кільцями відстані, режимом слідування за компасом та накладанням конуса поля зору.
- **Картки деталей** — Торкніться будь-якого об'єкта для повної інформації: реєстрація, оператор, маршрут (відправлення/призначення), висота, швидкість, курс, код squawk, джерело виявлення та рівень впевненості.
- **Журнал історії** — Постійна база даних усього, що ви ідентифікували, з можливістю пошуку та сортування.
- **Довідник дронів** — Вбудована база даних 30+ типів дронів з фотографіями, характеристиками та описами — від споживчих дронів DJI до військових БПЛА, таких як Bayraktar TB2, MQ-9 Reaper та Shahed-136. Переглядайте за категоріями, шукайте за назвою або переходьте прямо від невідомого виявленого дрона до ідентифікації того, що ви бачите.
- **Збагачення бекендом (опціонально)** — Опціональний Python API-сервер може додати назви авіаліній, реєстраційні номери та інформацію про маршрут понад те, що вже вбудовано. Додаток повністю працює без нього — фотографії літаків та довідник дронів включені в APK.

## Як це працює

### Джерела виявлення

| Джерело | Що виявляє | Дальність | Впевненість |
|---------|-----------|-----------|-------------|
| **ADS-B** | Комерційні та GA літаки з транспондерами | ~250 NM | Дуже висока (0.95) |
| **Remote ID (BLE)** | Дрони, сертифіковані FAA (250г+) | ~300м | Висока (0.85) |
| **WiFi SSID** | DJI, Skydio, Parrot, бюджетні дрони | ~100м | Середня (0.3-0.85) |
| **Візуальне (ML Kit)** | Все видиме в камері | Пряма видимість | Змінна |

### Злиття сенсорів та AR

Додаток поєднує дані акселерометра, магнітометра та гіроскопа, щоб точно визначити, куди спрямований телефон. Позиції літаків (широта/довгота/висота) проєктуються на вид камери за допомогою обчислень відстані за формулою гаверсинусів та геометрії поля зору камери.

**Гібрид ARCore + компас**: ARCore забезпечує відмінне відстеження, коли камера бачить наземні об'єкти, але має труднощі при спрямуванні на безхмарне небо — саме тоді, коли це найбільш потрібно. Додаток автоматично перемикається на орієнтацію за компасом-математикою, коли ARCore втрачає відстеження, забезпечуючи безперервні мітки за будь-яких умов.

## Початок роботи

### Передумови

- **Android Studio** Hedgehog (2023.1.1) або новіше
- **JDK 17**
- **Android SDK 35** (compileSdk) з minSdk 26 (Android 8.0+)
- Android-пристрій з GPS, компасом та камерою (емулятор працює для списку/карти, але не для AR)

> **Акаунти та реєстрація не потрібні.** Додаток підключається безпосередньо до безкоштовних публічних API ADS-B — без налаштування бекенду, без API-ключів, без акаунтів.

### Швидкий старт — лише Android

Додаток підключається безпосередньо до безкоштовних публічних API ADS-B — без налаштування бекенду або акаунтів. Фотографії літаків та довідник дронів вбудовані в APK. Щоб просто зібрати та запустити:

```bash
cd android
./gradlew assembleDebug
# Встановити на підключений пристрій:
adb install app/build/outputs/apk/debug/app-debug.apk
```

Або завантажте останній готовий APK з [**GitHub Releases**](https://github.com/lnxgod/friendorfoe/releases).

### Налаштування бекенду (опціонально — вмикає збагачення)

Бекенд додає фотографії літаків, назви авіаліній, пошук реєстрації та інформацію про маршрут. **Потрібен Python 3.11+ та опціонально Redis для кешування.**

```bash
cd backend

# Створити віртуальне середовище
python -m venv .venv
source .venv/bin/activate  # або .venv\Scripts\activate на Windows

# Встановити залежності
pip install -r requirements.txt

# Скопіювати конфігурацію середовища
cp .env.example .env
# Відредагуйте .env, якщо хочете додати облікові дані OpenSky (опціонально, збільшує ліміти)

# Запустити сервер
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**З Docker** (включає Redis + PostgreSQL):

```bash
cd backend
docker compose up
```

API буде доступний за адресою `http://localhost:8000`. Перевірка стану: `http://localhost:8000/health`

### Підключення додатку до бекенду

Оновіть URL бекенду в мережевій конфігурації Android-додатку, щоб вказати на IP-адресу вашого сервера.

## Джерела даних та атрибуція

Friend or Foe є 100% безкоштовним — кожне джерело даних нижче є відкритим і не потребує **API-ключів або платних акаунтів**.

### Дані позицій літаків ADS-B

Додаток використовує трирівневий ланцюжок резервування, щоб ви завжди отримували дані про літаки, навіть якщо одне джерело недоступне:

| Пріоритет | Джерело | Кінцева точка API | Що надає | Автентифікація |
|-----------|---------|-------------------|----------|----------------|
| 1-й | **[adsb.fi](https://adsb.fi)** | `opendata.adsb.fi/api` | Позиції літаків у реальному часі, позивні, висота, швидкість, курс, коди squawk | Безкоштовно, без ключа |
| 2-й | **[airplanes.live](https://airplanes.live)** | `api.airplanes.live` | Ті самі дані, що й adsb.fi (сумісний формат ADSBx v2) | Безкоштовно, без ключа |
| 3-й | **[OpenSky Network](https://opensky-network.org)** | `opensky-network.org/api` | Вектори стану ICAO — позиція, швидкість, курс, вертикальна швидкість | Безкоштовно, без ключа (опціональний акаунт для вищих лімітів) |

Усі три джерела надають живі дані транспондерів ADS-B з мережі приймачів по всьому світу. Додаток запитує за GPS-координатами та радіусом і автоматично переходить до наступного джерела при збої або тайм-ауті.

### Збагачення та допоміжні дані

| Джерело | Що надає | Автентифікація |
|---------|----------|----------------|
| **[Planespotters.net](https://planespotters.net)** | Фотографії літаків за ICAO hex-кодом — показуються на картках деталей | Безкоштовно, без ключа |
| **[Open-Meteo](https://open-meteo.com)** | Поточна погода (хмарність, швидкість вітру, умови) — використовується для налаштування параметрів виявлення | Безкоштовно, без ключа |
| **[OpenStreetMap](https://openstreetmap.org)** | Тайли карти через OSMDroid для 2D-виду карти | Безкоштовно, без ключа |

### Виявлення на пристрої (без зовнішніх API)

Ці методи виявлення працюють повністю на телефоні без мережевих запитів:

- **FAA Remote ID (Bluetooth LE)** — Сканує сигнали сертифікованих дронів у радіусі ~300м
- **Зіставлення WiFi SSID** — Ідентифікує DJI, Skydio, Parrot та 100+ інших виробників дронів за WiFi-шаблонами
- **Візуальне виявлення (ML Kit)** — Розпізнавання об'єктів на основі камери на пристрої
- **Військова класифікація** — Шаблони позивних, коди squawk та база даних операторів (все вбудовано локально)
